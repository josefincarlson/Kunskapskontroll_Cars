## Kunskapskontroll – Machine Learning
Detta repository innehåller vår inlämning för kunskapskontrollen i kursen Machine Learning.  
Uppgiften går ut på att använda maskininlärning för att förutsäga bilpriser baserat på ett dataset.

## Projektstruktur
- Kunskapskontroll_Cars_Grand_Finale.ipynb  
  Slutlig notebook som utgör inlämningen

- Övriga notebooks  
  Tidigare individuella eller experimentella notebooks som använts under arbetets gång

- car_price_dataset.csv  
  Dataset som används i projektet

## Projektinnehåll
I notebooken arbetar vi med följande steg:

- Inläsning och analys av data  
- Preprocessing och hantering av variabler  
- Träning av flera olika maskininlärningsmodeller  
- Utvärdering av modeller med RMSE  
- Stabilitetstest med olika random states  
- Val av slutlig modell  
- Slutsats och reflektion  

## Modellval
Efter jämförelse av flera modeller valdes Ridge Regression som slutlig modell.  
Modellen gav lågt och stabilt RMSE och bedömdes därför ha god generaliseringsförmåga.

## Köra notebooken
Notebooken körs uppifrån och ner.  
Alla filvägar är relativa, vilket gör att notebooken fungerar på olika datorer.
